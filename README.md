# UI-Automation-Framework_Python-Selenium-Pytest

## About
This is a complete Enterprise UI Automation framework that shows end-to-end testing of an ecommerce website.
It also incorporates Continuous Integration (CI) so when there is a push in the repository, it will trigger the pytest.
Running the pytest also generates test report and saves it in reports directory.

## Pre-Requisites
Python 3.14

## Skills
- Python
- Selenium
- Pytest
- Logging

## How it Works

Clone the repository, activate virtual environment, install the required packages listed in requirements.txt, then finally run the pytest
